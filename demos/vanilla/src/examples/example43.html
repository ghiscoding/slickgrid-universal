Demonstrates the optional @slickgrid-universal/web-mcp package (SlickWebMcpService), which exposes the grid as Model Context Protocol (MCP) tools so that AI assistants can read and manipulate the grid via natural language. The buttons below simulate what an LLM would call — in a real WebMCP-capable browser the same methods are called automatically by the AI assistant via navigator.modelContext.
