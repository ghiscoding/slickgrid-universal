Example 49: Spreadsheet Drag-Fill code

Spreadsheet with drag-fill, hybrid selection model. Type a few values in the grid and then select those cells and use the bottom right drag handle spread the selection and auto-fill the values to other cells. Use onDragReplaceCells event to customize the drag-fill behavior. Use { enableSelection: true, selectionOptions: { selectionType: 'mixed' }} grid option to enable the new Hybrid Selection Model.
