Example 48: Hybrid Selection Model code

SlickHybridSelectionModel This Selection Model is an hybrid approach that uses a combination of the row or cell selections depending on certain conditions.
  • 1. clicking on the first column (id) will use RowSelectionModel because of our configuration of rowSelectColumnIds: ['id'] as the columns that will trigger row selection.
  • 2. clicking on the any other columns will use CellSelectionModel by default
