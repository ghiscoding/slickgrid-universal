Example 31 - Dynamically Create Grid from CSV / Excel import
see code

Allow creating a grid dynamically by importing an external CSV or Excel file import. The script will read the file and will consider the first row as the column header and create the column definitions accordingly, while the next few rows will be considered to be the dataset.
This demo currently uses the following static CSV list for demo purposes (it could also work with any imported CSV list)
First Name,Last Name,Age,Type
John,Doe,20,Student
Bob,Smith,33,Teacher
Jane,Doe,21,Student
