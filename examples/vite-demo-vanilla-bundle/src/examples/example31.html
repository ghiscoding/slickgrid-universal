Example 31 - Dynamically Create Grid from CSV / Excel import
see code

Allow creating a grid dynamically by importing an external CSV or Excel file import. This script will read the CSV file and will consider the first row as the column header and create the column definitions accordingly, while the next few rows will be considered to be the dataset. Note that this example was created with a CSV file but in your application you could allow uploading an Excel file.
