Example 32 - Column & Row Span
see code

This page demonstrates colspan & rowspan using DataView with item metadata.
Note: colspan & rowspan are rendered via a row/cell indexes, any operations that could change these indexes (i.e. Filtering/Sorting) will require you, it becomes your responsability, to implement proper logic to recalculate these indexes. This demo does not show this because it is up to you to decide what to do when the span changes shape (i.e. you default to 3 rowspan but you filter a row in the middle, how do you want to proceed?).
