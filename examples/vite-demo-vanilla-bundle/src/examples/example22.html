Example 22 - Row Based Editing (with Salesforce Theme)
see code

  • The Row Based Edit plugin allows you to edit either a single or multiple specific rows at a time, while disabling the rest of the grid rows.
  • Editedable rows, as well as modified cells are highlighted with a different color, which you can customize using css variables (see example22.scss )
  • Modifications are kept track of and if the cancel button is pressed, all modifications are rolled back.
  • If the save button is pressed, a custom "onAfterRowUpdated" callback is called, which you can use to save the data with your backend.
    NOTE: You can also combine this with e.g. Batch Editing like shown in Example 11
